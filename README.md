# Belajar Go (Golang) - Interaktif

Halaman ini adalah materi interaktif sederhana untuk belajar dasar-dasar Go.

File di repository:
- `index.html` - Halaman utama
- `assets/style.css` - Styling
- `assets/app.js` - Logika interaktif

Cara pakai:
1. Buka `index.html` di browser modern.
2. Pilih topik di sidebar. Kode akan muncul di editor.
3. Gunakan tombol "Copy", "Download .go", atau "Buka di Go Playground" (jika tersedia).
4. Untuk menjalankan lokal: simpan file .go lalu jalankan `go run nama.go`.

Catatan:
- Tombol "Buka di Go Playground" mencoba mengirim kode via form POST; bila Playground diblokir, gunakan tombol salin / unduh.
- Syntax highlighting disediakan oleh Prism.js via CDN.
